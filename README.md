YanProperty

YanProperty adalah platform properti yang menampilkan listing rumah dan apartemen secara terperinci dengan fokus pada kemudahan eksplorasi, filter pencarian yang relevan, serta lead capture untuk calon pembeli. Website live: https://yan-property.vercel.app/

Live Website

• URL: https://yan-property.vercel.app/

Fitur Utama

• Tampilan listing properti yang rapi dan responsif
• Pencarian dan filter sederhana (lokasi, harga, tipe properti, dsb.)
• Halaman detail properti dengan photos, deskripsi, dan kontak penjual
• Integrasi lead capture untuk mengumpulkan minat pembeli
• SEO dasar dan optimasi kecepatan halaman untuk pengalaman pengguna yang lebih baik

Cara Mulai (Local Development)

Prasyarat:

• Node.js (sesuaikan versi yang digunakan proyek Anda)
• Git (untuk kloning repositori)

Langkah-langkah:

1. Clone repo Anda:
  • git clone git@github.com:firmandhanii/YanProperty.git
2. Pergi ke folder proyek:
  • cd YanProperty
3. Pasang dependensi:
  • npm install
  • atau yarm install
4. Jalankan pengembangan lokal:
  • npm run dev
  • atau yarn dev

Catatan: Sesuaikan perintah jika proyek Anda menggunakan npm scripts yang berbeda.

Struktur Direktori (Contoh)

• src/ atau app/        - kode frontend utama
• public/                - aset statis
• pages/ atau src/pages  - rute halaman (untuk Next.js)
• components/            - komponen UI reusable
• styles/                - gaya global

Sesuaikan dengan struktur proyek Anda yang sebenarnya.

Cara Deployment

• Proyek ini dihosting di Vercel. Perubahan kode di GitHub akan memicu build dan deploy otomatis di Vercel (pastikan repository terhubung ke Vercel).
• Pastikan environment variables jika ada (mis. API keys) diatur di dashboard Vercel.
